# Phishing Awareness & Simulation System

## Overview
A web-based application developed as a final year project at the 
Federal University of Technology Owerri (FUTO), Department of 
Cybersecurity, 2024.

## Purpose
Designed to educate Nigerian university students on recognizing 
and preventing phishing attacks through interactive simulation 
and awareness training.

## Key Features
- Admin Login Page for managing the awareness system
- Awareness/Education Page with phishing prevention content
- Mock Phishing Simulation Form to test user susceptibility

## Results
- 120 student participants surveyed
- Reduced phishing susceptibility from 65% to below 40%
- 37.5% of participants scored above 90% on post-training quiz
